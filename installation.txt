Installation

Install Python3.9 from here manually.
#########Install project dependencies by running py -m pip install -r requirements.txt.
Run the commands python3.10 manage.py makemigrations and python3.10 manage.py migrate in the project directory to make and apply migrations.
Create superuser with python3.10 manage.py createsuperuser. This step is optional.
Run the command python3.10 manage.py runserver to run the web server.
Open web browser and goto 127.0.0.1:8000 url to start using the web application.
